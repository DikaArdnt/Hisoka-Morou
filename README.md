## Installing
```bash
$ git clone https://github.com/DikaArdnt/Hisoka-Morou
$ cd Hisoka-Morou
$ npm start / node .
```

## FOR TERMUX/UBUNTU/SSH USER

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
git clone https://github.com/DikaArdnt/Hisoka-Morou
cd Hisoka-Morou
npm install
```
